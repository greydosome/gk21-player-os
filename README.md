# GK21

GK21 is a personal Player Operating System for managing condition, diet, workout, goalkeeper performance, and AI coaching.

## Directory Policy

- `/home/rocky/gk21` : Application, source code, Git repository
- `/postgres` : PostgreSQL data, logs, backup, and DB operation scripts

## Stack

- Backend: FastAPI
- Database: PostgreSQL 17
- ORM: SQLAlchemy 2.x
- Migration: Alembic
- Frontend: React + TypeScript + Vite
- Web: Nginx
- App Mode: PWA
